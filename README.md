# NEiP
A NES emulator coded in Python, although most all of it is in Cython for speed and efficiency.

It's extremely broken though, so don't expect it to perform well and be accurate at most anything.

## How To Use
It's all pretty straight forward. Use the menu at the top of the window (top of the screen on MacOS) to select file options, emulator options, etc.

To use the SSTs put all of the SST json files in a folder called SSTs and then select the SST button from the menus.
